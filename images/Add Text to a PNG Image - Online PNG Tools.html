Text to png adder tool What is a text to png adder?
This tool adds text labels to PNG images. You can customize the color of the text, background color, size and position of the text box, text typeface, font size and make it bold or italic. Additionally can also import a custom font from a URL. Quick and easy!

Add text to the center of a PNG image
This example adds white text with a semi-transparent tint to the absolute center of a PNG picture. Omitting the label position and size stretches the text box to the full size of the image, and the text is automatically centered inside the box.
